# PictVideo Call

**Live demo:** https://pictvideo-call.onrender.com/

PictVideo Call is a simple real-time video calling application built using WebRTC, Node.js, Express, and Socket.io.
  
(Tech stack, features: e.g. WebRTC, React, Node/Express, deployed on Render.)

## How to run locally
1. git clone https://github.com/Siddheshc09/pictvideo_call.git
2. cd pictvideo_call
3. Install & run backend/client as described in their folders.

## Live Demo
Open the app: https://pictvideo-call.onrender.com/
